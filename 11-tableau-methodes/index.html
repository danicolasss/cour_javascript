Les principales méthodes liées aux tableaux

Filtrer un tableau

Définition

Filtrer un tableau consiste à créer et retourner un nouveau tableau contenant tous les éléments du tableau d'origine qui remplissent une condition déterminée par une fonction de callback.

TableauSource --- filtrer ---> TableauDestination

Méthode à utiliser : filter

Syntaxe

tableauDestination = tableauSource.filter( element => ...)

IMPORTANT

  • la taille de tableauDestination est <= taille de tableauSource
  • les éléments des 2 tableaux ont la même structure

Mapper un tableau

Définition

Mapper un tableau consiste à créer et retourner un nouveau tableau contenant tous les éléments du tableau d'origine sur lesquels on a appliqué une transformation déterminée par une fonction de callback.

TableauSource --- mapper ---> TableauDestination

Méthode à utiliser : map

Syntaxe

tableauDestination = tableauSource.map( element => ...)

IMPORTANT

  • la taille de tableauDestination est égale à taille de tableauSource
  • les éléments des 2 tableaux n'ont pas obligatoirement la même structure

Trier un tableau

Définition

Trier (sort) un tableau consiste à réorganiser les éléments suivant un ou plusieurs critères.
Le ou les critères peuvent être précisés par une fonction de callback. Si celle-ci n'est pas définie, alors le tri est exécuté en utilisant un critère par défaut.
Le tri est réalisé sur le tableau lui-même ! et renvoie le tableau !

TableauSource --- trier ---> TableauSource

Méthode à utiliser : sort

Syntaxe

  • tableauSource = tableauSource.sort() // Tri par défaut
  • tableauSource = tableauSource.sort( (a,b) => ...)// Fonction de comparaison

Tri d'un tableau dont les éléments sont des chaines ou des numériques

Par défaut, le tri s'effectue sur les éléments du tableau convertis en chaines de caractères et triés selon le code UTF-16 des caractères (les majuscules sont "avant" les minuscules, 12 est avant 2, etc...). Cette solution peut ne pas être satisfaisante.

Pour cela, on peut définir une fonction de comparaison que l'on va passer en argument à la fonction sort. L'algorithme général de cette fonction de callback est présenté ci-dessous : 

       
       (a, b) => {
            if (a est inférieur à b selon les critères de tri)
                return -1;
            if (a est supérieur à b selon les critères de tri)
                return 1;
            // a doit être égal à b
            return 0;
        }

Pour comparer des valeurs numériques, on peut simplifier la fonction de comparaison comme ci-dessous :

       
       (a, b) => {
           return a-b;
       }
       // On peut simplifier par :
       (a,b) => a-b

IMPORTANT

Pour effectuer un tri descendant, il suffit d'inverser le critère de comparaison !
